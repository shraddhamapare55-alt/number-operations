JavaScript Number Operations
This project performs different operations on a given number using JavaScript these are:
Prints the first n numbers
Prints the table of n
Finds the sum of digits
Checks whether the number is prime
Prints all factors of the number
Checks whether the number is an Armstrong number

Variables Used
n = Number to perform operations on
sum = Sum of digits
temp = Temporary variable for calculations
digits = Number of digits
armstrongSum = Sum used for Armstrong check
prime = Prime number status

How to Run

Open the project folder.
Open `index.html` in a web browser.
Press `F12` to open Developer Tools.
Open the **Console** tab.
View the output.
